## ディレクトリ構成
```text
├─.next
├─messages　          # 多言語対応(日本語,英語)のための言語ファイルを格納
├─node_modules
├─public              # 静的ファイルを格納
│  ├─demo
│  ├─images
│  │  ├─others
│  │  └─projects 
│  └─sounds
└─src　　　　　　　     
    ├─components　　　　　
    │  ├─about
    │  ├─elements　　　#共通して使用するコンポーネント
    │  │  ├─background
    │  │  ├─button
    │  │  ├─frame
    │  │  └─text
    │  ├─hero　　
    │  ├─layouts　　　 #ヘッダーやフッターなどレイアウト関連のコンポーネント
    │  ├─others
    │  ├─projects
    │  ├─services
    │  └─skills
    ├─pages    
    └─styles          #Tailwind CSS,Framer Motion関連のファイルを格納
```
